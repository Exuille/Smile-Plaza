<h2 id="about-the-project">üè• About The Project </h2>
<p align="justify">
  &nbsp;&nbsp;&nbsp;&nbsp;<a href="#top"><strong>Smile Plaza Dental Center Appointment System</strong></a> is a web application designed to streamline the appointment scheduling process for both customers and administrative staff, particularly dentists. This system empowers customers to effortlessly book appointments online, while providing administrative access for managing these appointments efficiently. This Dental Appointment System is based on the traditional appointment system of Smile Plaza Dental Center located at Candelaria, Quezon.
</p>

## üöÄ How to Setup

### üìù Prerequisites

Make sure you have the following installed:

- Visual Studio Code (VSCode)
- Node.js v22.11.0 or higher
- npm v10.9.0 or higher
- MongoDB (local or cloud)

---

### üîÑ Clone the Repository

Clone the project using this URL:

```
https://github.com/Exuille/Smile-Plaza.git
```

Or use the **"Open in GitHub Desktop"** button on the repository page.

---

## üåê Frontend Setup

1. Open a terminal and navigate to the frontend folder:

```
cd frontend
```

2. Create a `.env` file in the `frontend` folder and add the following content:

```
VITE_REACT_APP_SITE_KEY=YOUR_SITE_KEY_HERE
```

3. Install the required dependencies:

```
npm install
```

Or:

```
npm i
```

4. Start the development server:

```
npm run dev
```

---

## üõ† Backend Setup

1. Open a new terminal or press `CTRL + SHIFT + \`` in VSCode.

2. Navigate to the backend folder:

```
cd backend
```

3. Install the backend dependencies:

```
npm install
```

Or:

```
npm i
```

4. Create a `.env` file inside the `backend` folder with the following content:

```
URI=YOUR_MONGODB_SERVER_HERE
PORT=3001
JWT_SECRET=YOUR_JWT_SECRET_HERE
JWT_EXPIRES_IN=1h
SITE_SECRET=YOUR_SITE_SECRET_HERE
```

> ‚ö†Ô∏è **Do not change the PORT.** The server uses port 3001 by default.

5. Start the backend server:

```
npm start
```
